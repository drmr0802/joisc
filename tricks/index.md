## bitmask

- $x \oplus (x + 1) \oplus (x + 2) \oplus (x + 3)$ 有性质
- 条件 $\oplus A = 0$ 可以变为，$\oplus B = \oplus C, B \cup C = A$。
- 若 $\oplus L = \oplus R$。则设 $A = L \backslash B, B = L \cap R, C = R \backslash B$，有 $\oplus A = \oplus C$。

### 组合恒等式

$$
\binom{n}{m} = \binom{n - 1}{m} + \binom{n - 1}{m - 1}
$$

$$
\sum_{j = 0}^{i} \binom{i}{j} = 2^i
$$

$$
\sum_{i = 0}^{k} \binom{n}{i} \binom{m}{k - i} = \binom{n + m}{k}\\
\sum_{a + b = k} \binom{n}{a} \binom{m}{b} = \binom{n + m}{k}\\
\sum_{a + b = k} \binom{a}{c} \binom{b}{d} = \binom{k + 1}{c + d + 1}
$$

$$
\sum_{i = 0}^{n} (-1)^i \binom{n}{i} = 0
$$

$$
\sum_{k = 0}^{n} k \binom{n}{k} = n 2^{n - 1}\\
\sum_{k = 0}^{n} k^2 \binom{n}{k} = n (n + 1) 2^{n - 2}
$$

$$
\sum_{i = 0}^{n} \binom{i}{k} = \binom{n + 1}{k + 1}
$$

### 括号序列

假如 $[A, B], [C, D]$ 都是合法括号序列，那么 $[A, C - 1], [C, B], [B + 1, D]$ 也是合法括号序列。（$A < C < B < D$）
